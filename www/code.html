Trabalho de Conclusão de Curso

Undergraduate Thesis

Direção Autônoma de Robôs Móveis por Classificação de Imagens Através de Aprendizado Discriminativo de Redes Soma-Produto

Autonomous Mobile Robot Driving Through Image Classification Using Discriminative Learning of Sum-Product Networks

Aluno:Student: Renato Lui Geh
Orientador:Advisor: Denis Deratani Mauá

Código

Serão implementadas as versões soft e hard do algoritmo de aprendizado discriminativo de Gens-Domingos. O código escrito para este TCC será dividido em duas partes. A primeira será a implementação do algoritmo para classificação de imagem. A segunda será a aplicação dele à tarefa de direção autônoma de robôs móveis. Todo o código escrito será livre e aberto.

Aprendizado Discriminativo

O código se baseará no artigo:

Robert Gens, Pedro Domingos. "Discriminative Learning of Sum-Product Networks". Advances in Neural Information Processing Systems (NIPS 2012). Ed. 25.

A implementação será disponibilizada em GoSPN, uma biblioteca de inferência e aprendizado de redes soma-produto. A implementação será escrita em Go.

Aplicação em Robôs Móveis

A implementação será baseada no artigo:

Bruno Massoni Sguerra, Fabio G. Cozman. "Image Classification Using Sum-Product Networks for Autonomous Flight of Micro Aerial Vehicles". Brazilian Conference on Intelligent Systems (BRACIS 2016). Ed. 5.

O código para a aplicação em robôs móveis estará disponível em GoBot. Serão usadas as linguagens Go para inferência e classificação, e Java para a direção do robô móvel. Para os testes baseline, será usado Python.

Code

Both "soft" and "hard" versions of Gens-Domingos' discriminative learning algorithm will be implemented. We will divide our code into two parts. The first part refers to the implementation of the discriminative learning algorithm for image classification. The second will be code written for the autonomous driving of mobile robots task. All code will be free and open-source.

Discriminative Learning

We will base our code on the following article:

Robert Gens, Pedro Domingos. "Discriminative Learning of Sum-Product Networks". Advances in Neural Information Processing Systems (NIPS 2012). Ed. 25.

The implementation will be part of the GoSPN framework for inference and learning of sum-product networks. It will be written in Go.

Application to Mobile Robots

Our implementation will be based on the article:

Bruno Massoni Sguerra, Fabio G. Cozman. "Image Classification Using Sum-Product Networks for Autonomous Flight of Micro Aerial Vehicles". Brazilian Conference on Intelligent Systems (BRACIS 2016). Ed. 5.

We will make our code available at GoBot. We shall use Go for inference and classification and Java for the robot's driving functions. Baseline tests will be written in Python.
